üé¨ Embed Anime Player

Embed anime players directly on your website using iframes.

GET /anime/:anilistId/:episode/:language
anilistId The Anilist ID of the anime (required)
episode Episode number (required)
language Audio language: sub, dub, hindi (default: sub)

Embed Code Example:

<iframe 
  src="https://anisnfdf.vercel.app/anime/21/1/sub"
  frameborder="0" 
  scrolling="no" 
  allowfullscreen
  style="width: 100%; height: 400px;"
></iframe>
